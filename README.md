## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/Denizkaya392/date-find.git
```

2. Install dependencies:
```bash
cd date-find
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
